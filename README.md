# NLP Intelligence Lab

A personal research lab for exploring NLP concepts beyond the basics.
Not a course. Not tutorials. Experiments, intuitions, and findings.

---

## What is this?

A structured playground for:
- Understanding how text representations work
- Visualizing embeddings and similarity
- Running small experiments to build intuition
- Documenting findings like a research log
